# Tade Packaging Website

A modern, responsive, and SEO-friendly website for an industrial corrugated box manufacturing company.

## Tech Stack
- **React (Vite)**
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **React Router Dom** (Routing)
- **React Helmet Async** (SEO)

## Features
- **Dynamic Routing**: Individual product pages using slugs.
- **Product Filter**: Filter products by category (Standard, Printed, Custom, Industrial).
- **Inquiry Forms**: Lead generation forms on product detail and contact pages.
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.
- **SEO Ready**: Dynamic meta tags for every page.
- **WhatsApp Integration**: Floating contact button for instant inquiries.

## Project Structure
```
src/
 ├── components/    # Reusable UI components (Navbar, Footer, etc.)
 ├── pages/         # Page components (Home, About, Products, etc.)
 ├── layouts/       # Page wrappers
 ├── data/          # Mock data (Products catalog)
 ├── assets/        # Static files
 └── App.jsx        # Routing configuration
```

## How to Run
1. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
